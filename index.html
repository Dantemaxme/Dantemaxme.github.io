2. 自动部署

  • 首先，安装一个可爱的vs code(●’◡’●)
  • 然后，安装shelljs（这是一个自动加载js脚本的模块）
  • 接着，hexo的根目录（就是source并行目录）下，创建scripts文件夹
    如果有，直接打开就好
    创建一个js脚本，内容为 D:\ANACONDA\Microsoft VS Code Insiders\Code - Insiders.exe 更换自己的文件目录就好啦
  • 最后，可以将VScode设为md文件的自动打开exe
